Mảng 1 Chiều

1. Giới thiệu

Mảng 1 chiều là tập hợp các phần tử cùng kiểu dữ liệu được lưu liên tiếp trong bộ nhớ.

2. Khai báo

int a[1000];

a[i] là phần tử thứ i của mảng.

3. Nhập xuất mảng

for(int i=0;i<n;i++) cin>>a[i]; for(int i=0;i<n;i++) cout<<a[i]<<" ";

4. Duyệt mảng

for(int i=0;i<n;i++) { cout<<a[i]; }

Độ phức tạp: O(n)

5. Tìm kiếm tuyến tính

int pos=-1; for(int i=0;i<n;i++) { if(a[i]==x) { pos=i; break; } }
Thuật toán ĐPT
Linear Search O(n)
Binary Search O(log n)

6. Sắp xếp

sort(a,a+n);

Sử dụng hàm sort của STL.

7. Các bài toán thường gặp

  • Tìm max/min
  • Đếm số chẵn lẻ
  • Tính tổng
  • Tìm phần tử lớn thứ 2
  • Kiểm tra đối xứng
  • Dãy con liên tiếp
  • Prefix Sum
  • Two Pointers
  • Sliding Window

8. Bài tập luyện tập

  1. Tính tổng mảng
  2. Tìm số lớn nhất
  3. Đếm số nguyên tố
  4. Sắp xếp tăng dần
  5. Tìm phần tử xuất hiện nhiều nhất
  6. Tìm đoạn con có tổng lớn nhất
